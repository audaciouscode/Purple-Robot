Plays the audio notification specified by tone on the device.

Arguments

tone - Name of the notification tone to play.

loops (Optional) - Boolean value specifying whether to repeat playing the tone in a loop. Default is false.

See PurpleRobot.stopPlayback for details about canceling a looping tone.

Return Values

None

Discussion

The names and availability of specific tones will vary by device and manufacturer.

Tones available on vanilla Android 4.x devices:

  • Hydra
  • Machina
  • Orion
  • Pegasus
  • Perseus
  • Pyxis
  • Radiation
  • Robots for Everyone
  • Scarabaeus
  • Sceptrum
  • Solarium
  • Themos
  • UrsaMinor
  • Zeta

Tones available on vanilla Android 5.x devices:

  • Callisto
  • Dione
  • Ganymede
  • Luna
  • Oberon
  • Phobos
  • Robots for Everyone
  • Sedna
  • Spagnola Orchestration
  • Titania
  • Triton
  • Umbriel
  • Zeta

Tones bundled with Purple Robot:

  • CRcv
  • Csnd
  • Error
  • New
  • Off
  • On

These tones originated as part of the Tokyo Train Station Adium plugin.

JavaScript Example

PurpleRobot.playTone('CRcv');

PurpleRobot.playTone('CSnd', true);

Scheme Example

(pr-play-tone "CRcv")

(pr-play-tone "CSnd" #t)
