Generates a synthetic probe reading and transmits it through the Purple Robot data processing facilities.

Arguments

name - String identifying the reading.

value - Value of the reading. value may consist of a string, number, list, or object that can be expressed with JSON-compatible types.

priority (Optional) - Number specifying the priority that the rest of the system should afford this reading. For example, readings with priorities will be transmitted earlier by some DataUploadPlugins than readings with no priority specified.

Return Values

None

JavaScript Example

PurpleRobot.emitReading("myFavoriteNumber", 7);

PurpleRobot.emitReading("myFavoriteColor", "blue", 1);

PurpleRobot.emitReading("myFavoriteFoods", [ "pizza", "pancakes", "cheese" ]);

PurpleRobot.emitReading("myFavoriteCar", { "mfgr": "Ford", "model": "Mustang", "engine": "5.0L V8" }, 20);
