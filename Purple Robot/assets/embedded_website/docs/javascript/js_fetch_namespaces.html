Returns an array of namespaces used in the internal storage system.

Arguments

None

Return Values

A string array containing namespace values.

JavaScript Example

var namespaces = PurpleRobot.fetchNamespaces();

console.log(namespaces.length + " namespace(s) are currently in use.");

for (var i = 0; i < namespaces.length; i++)
{
    console.log("   " + namespaces[i]);
}
